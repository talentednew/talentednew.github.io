1.3 mv

  1. The purpose of mv command moves, or renames file and directories on your file system
  2. The basic syntax is as followed: mv options -T source destination
  3. Examples
    mv myfile.txt myfiles
    Move the file myfile.txt into the directory myfiles. If myfiles is a file, it will be overwritten.
    mv myfiles myfiles2
    If myfiles is a file or directory, and myfiles2 is a directory, move myfiles into myfiles2.
    mv myfile.txt ../
    Move the file myfile.txt into the parent directory of the current directory.
    mv -t myfiles myfile1 myfile2
    Move the files myfile1 and myfile2 into the directory myfiles.
